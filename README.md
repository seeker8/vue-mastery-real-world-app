# real-world-vue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Install JSON Server
```
npm i -g json-server
```

### Start JSON server
```
json-server -d 1500 db.json
```

See [Configuration Reference](https://cli.vuejs.org/config/).
